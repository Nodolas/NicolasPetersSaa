# How to Start the Website

## Option 1 — Double-click (easiest)
Double-click `start.bat` in the project folder.
The site opens automatically in your browser with live-reload.

## Option 2 — Terminal
```
npm start
```
Run this once inside the project folder.
If it's your first time, run `npm install` first.

## Option 3 — No Node.js at all
Open `index.html` directly in your browser (file://).
No live-reload, but works without any tools installed.

## First-time setup (only needed once)
```
npm install
```
